# 📚 AI Study Buddy

AI-powered learning assistant built with Streamlit and Groq API.

## Features
- 🔍 Explain Complex Topics
- 📝 Summarize Study Notes
- ❓ Generate Quiz Questions
- 📇 Create Flashcards
- 💾 Save & Organize

## Quick Start

### Install
```bash
pip install -r requirements.txt
```

### Run
```bash
streamlit run study_buddy.py
```

Visit http://localhost:8501

## Requirements
- Python 3.8+
- Streamlit
- Groq API
